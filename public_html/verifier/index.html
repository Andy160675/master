Sovereign Verifier Surface

Status: presentation-only

Posture: FREEZE (Class 2 surface)

What this is

This site serves static artifacts for transparency and demonstration. It is not an authority-bearing verifier-of-record.

What this is not

  • No operational verification
  • No trust claims
  • No server-side evidence generation

Demo artifacts (Phase 2A)

  • demo/README.txt
  • demo/valid.json
  • demo/invalid.json

Manifests (tamper test)

Manifests are intended to be uploaded via SSH/SFTP only and retrieved via SFTP/SCP for offline verification.

  • manifests/ (directory)
  • SOVEREIGN_MANIFEST.txt
  • SOVEREIGN_MANIFEST.sha256

Integrity model

Integrity is defined solely by SHA-256 hashes verified offline from retrieved files. HTTP is not a verification channel.
